# unarr Homebrew tap

Homebrew tap for [unarr](https://unarr.app) — the self-hosted media manager, player and debrid agent CLI.

## Install

```sh
brew install unarr-app/tap/unarr
```

This pulls the latest signed release from [`Unarr-app/unarr-cli`](https://github.com/Unarr-app/unarr-cli).

## Update

```sh
brew upgrade unarr
```

The formula is kept in sync with each CLI release.
